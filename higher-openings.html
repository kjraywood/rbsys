17.2. Four-card major & long diamonds

In other situations when there are two possible trump suits, and two bids available to set the trump suit, we use the following:

  • If one of the available bids is one of the possible trump suits, then bidding that suit is natural, and a new suit sets the other as trump.
    e.g.  Second-suit slam-try after a major-suit raise.

  • If neither of the two available bids is a possible trump suit, then we use lower-for-lower.
    e.g.  1NT-3♦ = both minors, GF  ➤  3♥ ⇒ ♣, 3♠ ⇒ ♦

So this situation is an instance of the first case with diamonds as the primary suit.

17.3. Interference

  Unusual scheme

Many use double (or redouble) to show a bust, and pass as a generic positive. The lack of shape info leads to awkward auctions. We use double to show values without length in the bid suit. Thus pass is either a bust or a trap hand. This is similar to what strong one-clubbers do.

17.3.1. Vs double

 ●  redouble   = values but not ♣-length
 ●  pass = bust or ♣-length

17.3.2. Vs overcall

 ●  double   = values but not length in bid suit
 ●  pass = bust or length in bid suit

This is similar to how we counter overcalls of our strong 1NT opening.

17.3.3. Subsequent bidding

Opener’s double or redouble is take-out unless responder has shown positive values with a double or penalty pass.
